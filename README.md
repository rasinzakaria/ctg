# CTG Brands Homepage Recreation

This is a recreation of the CTG Brands homepage (https://www.ctgbrands.com/).

## Files Structure

- `index.html` - Main HTML structure
- `styles.css` - All CSS styling
- `script.js` - JavaScript for interactivity
- `README.md` - This file

## Features

✅ **Header & Navigation**
- Announcement banner (dismissible)
- Logo
- Main navigation with mega menus
- Search functionality
- Shopping cart icon
- Login/Retailer links
- Language selector

✅ **Hero Section**
- Large background image
- Overlay text box with call-to-action

✅ **Content Sections**
- Trending Categories (Seasonal & Everyday collections)
- Home Decor (3 brand cards with overlays)
- Become a CTG Partner section
- Trending Brands (Luciano & Bodico)
- Our Brands carousel
- Featured Images (True North & Easter)

✅ **Footer**
- Four-column layout
- Links organized by category
- Email subscription form
- Social media icons
- Scroll to top button

✅ **Responsive Design**
- Mobile-friendly navigation
- Responsive grid layouts
- Touch-friendly interactions

## How to Use

1. Open `index.html` in a web browser
2. No build process required - it's a static HTML page

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive design works on mobile, tablet, and desktop

## Notes

- All images are loaded from the original CTG Brands CDN
- Fonts are loaded from Google Fonts
- Icons use Font Awesome and Ionicons
- The page is fully functional but uses placeholder links for navigation

## Customization

You can customize:
- Colors in `styles.css`
- Content in `index.html`
- Interactive behavior in `script.js`
